# Finanzas_basicas
Es una aplicacion pensada para celular, que perdure en el tiempo, con posibilidad de hacer copias de seguridad manuales.
# Funciones a futuro
se iran añadiendo funciones en cuanto se me de ideas para ello.
# Funciones actuales
calendario para registrar acciones financieras. 
pestaña de resumen para ver el dinero que se posee y en que partes se ha ido.
graficos de manera pastel para los egresos registrados.
funcion de filtrado basico.
funciones de deudas basicas.
tiene iconos para hacer mas entendible que estas leyendo
modo oscuro por que el modo blanco es terrible (posible modo azul o gris en un futuro)
